
### Build an API server implementing our OpenAPI spec for food ordering API in Go.

Go version used: 1.25

API Spec Published at https://orderfoodonline.deno.dev/public/openapi.yaml

### How to run

```
make run
```

### Run unit tests

```
make test
```
